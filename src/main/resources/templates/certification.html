Recebemos o seu formulário!

Veja a seguir os passos restantes para obter o selo de certificacão zeroWaste!

1. Aplicação

  • A pessoa responsável pelo condomínio preenchera os dados na aplicação para enviar o formulário.

2. Auditoria

  • O sistema avaliara a solicitação e marcara uma data junto a pessoa responsável para realizar a auditoria no local, a modo de atestar a eficiência do condomínio seguindo os critérios da zeroWaste.

3. Certificação

  • Caso o informe da auditoria seja positiva, será concedido a certificação zeroWaste para o condomínio, com validade por um ano.
